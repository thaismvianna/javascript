Operadores

Aritméticos: +(adição); -(subtração); *(multiplicação); /(divisão); %(operador de resto da divisão sendo ela inteira); **(exponenciação)

Atribuição: Atribuição simples (=): Atribui o valor a direita e a variável a esquerda.
Atribuição com adição (+=): Adiciona o valor a direita ao valor atual da variável e atribui o resultado.
Atribuição com subtração (-=): Subtrai o valor a direita do valor atual da variável e atribui o resultado.
Atribuição com multiplicação (*=): Multiplica o valor atual da variável pelo valor a direita e atribui o resultado.
Atribuição com divisão (/=): Divide o valor atual da variável pelo valor a direita e atribui o resultado.
Atribuição com resto (%=): Calcula o resto da divisão do valor atual da variável pelo valor a direita e atribui o resultado.
Atribuição com exponenciação (**=): Eleva o valor atual da variável a potência do valor à direita e atribui o resultado.

Relacionais: Os operadores relacionais comparam valores e retornam true ou false.
== (Igual a): Verifica se os valores são iguais. Ex: 5 == "5" → true
=== (Estritamente igual a): Verifica valor e tipo. Ex: 5 === "5" → false
!= (Diferente de): Verifica se os valores são diferentes. Ex: 5 != "6" → true
!== (Estritamente diferente de): Verifica valor ou tipo. Ex: 5 !== "5" → true
> (Maior que): Verifica se o valor da esquerda é maior. Ex: 10 > 5 → true
< (Menor que): Verifica se o valor da esquerda é menor. Ex: 3 < 2 → false
>= (Maior ou igual a): Verifica se é maior ou igual. Ex: 5 >= 5 → true
<= (Menor ou igual a): Verifica se é menor ou igual. Ex: 4 <= 3 → false
== e != ignoram o tipo (só comparam valores).
=== e !== exigem tipo e valor iguais.
>, <, >=, <= comparam números/strings.

Lógicos: Os operadores lógicos combinam expressões e retornam true ou false.
&& (conjunção: AND → E): true somente se TODAS as condições forem verdadeiras. Ex: (5 > 3) && (2 < 4) → true (porque as duas são verdade).
|| (disjunção: OR → OU): true se PELO MENOS UMA condição for verdadeira. Ex: (10 == 5) || (3 !== "3") → true (a segunda é verdade).
! (negação: NOT → NÃO): Inverte o valor: true vira false e vice-versa. Ex: !(10 > 5) → false (porque 10 > 5 é true, e o ! nega).
&& = "E" → Só passa se todas forem verdade.
|| = "OU" → Passa se pelo menos uma for verdade.
! = "NÃO" → Inverte o resultado (útil para checar o oposto).

Ordem de precedencia quando tem operadores aritmeticos, relacionais e logicos em uma mesma expressão:
Primeiro faz os operadores aritmeticos, depois os relacionais, depois os lógicos.
Ex: var a = 5
var b = 8
a > b && b % 2 == 0
false

Outra ordem importante é: caso tenha em uma mesma expressão o &&(E), ||(OU) e !(NÃO), se faz primeiro !, depois && e depois o ||

Ternário: É a interrogação e os dois pontos dentro de uma mesma expressão. É ternario pois tem três partes. Ex: teste(bloco1) ? true(bloco2) : false(bloco3)
?(interrogação): "Se" (testa a condição).
: (dois pontos): "Senão" (o que retornar se for falso).
Em ordem de precedencia o ternário é sempre o último a ser feito.
